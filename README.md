# Adding Skills
Below you can find the information how to add Skills to corresponding AI agent

## Claude Desktop (MacOS)
To add Skills in Claude Desktop follow these steps: 
1. Download corresponding **Skill.md** file to your PC 
2. Open **Claude Desktop**
3. Go to **Settings**. Select **Capabilities**
4. Scroll down to the bottom and click **Customize**

   <img width="1081" height="513" alt="image" src="https://github.com/user-attachments/assets/411705db-a60c-4581-b59c-9ad84331a707" />

6. Click **Plus** > **Create skill** > **Upload a Skill**

   <img width="803" height="347" alt="image" src="https://github.com/user-attachments/assets/d943e81a-fb1e-4392-887d-0976fd285e43" />

8. Drop or select the file you downloaded in Step 1.
9. Once the Skill is uploaded, you can use it in any chat using **/** command
